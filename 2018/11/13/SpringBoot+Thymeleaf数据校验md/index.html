SpringBoot+Thymeleaf数据校验

Posted on 2018-11-13 | In SpringBoot |

1SpringBoot对表单数据校验的技术特点

SpringBoot中使用了Hibernate-validate校验框架
Hibernate-validate 是start-web依赖中的jar，不用额外注入

非空校验@NotBlank

pojo类

public class Users {
@NotBlank //非空校验

private String name;

controller

/**

 * 完成用户添加
 *@Valid 开启对Users对象的数据校验
 *BindingResult:封装了校验的结果
 */
@RequestMapping("/save")
public String saveUser(@Valid Users users,BindingResult result){
    if(result.hasErrors()){
        return "add";
    }
    System.out.println(users);
    return "ok";
}

BindingResult result 是SpringMvc中的类 

用户姓名：<input type="text" name="name"/><font color="red" th:errors="${users.name}"></font><br/>

会与 th:errors=”${users.name}” 联动 ，会在font标签展示错误信息

使用BindingResult需要在showPage方法中传入User类对象 ，要求参数对象的变量名必须是对象的类名的全称首字母小写。

@RequestMapping(“/addUser”)
public String showPage( Users users){
return “add”;

}

建议采用上述方法，如果参数名需要做改变

@ModelAttribute(“aa”) 改变在springMvc中BindingResult 的Model的Key值

表示当前传递的对象的key为aa
例如：th:errors=”${aa.password}”

开启@Valid对实体类对象数据检验

例如：在Controller方法中 传参位置加入@Valid Users users

注意

@NotBlank判断字符串去掉首位空格是否为Null，如果校验数据为” “这样的空串是不会error的
如果判断是否为空串需要@NotEmpty

其他注解

@Length 判断字符串长度

例如：

@Length(min=1,max=10)

private String a;

@Min 判断数值最小值
@Min(value=10)
private Interger i;
@Max 判断数值最大值
@Email: 判断邮箱是否合法
